# Class Activities Repository

## What? 
This repository contain all in-class materials and starter projects for the 8-week program. The starter code for the activities will be provided here moving forward. Starting with the end of this week, any code related solutions will be added for the previous classes for that week. For example, the solutions for this activity will be pushed by Friday March 6th.

## How?
1. Students should clone this repository at the start
    ```git clone https://github.com/chadhindsight/Swift-Activities-BMCC-) ```
2.  Open the project folder in Xcode.
3.  At the beginning of each class, pull the latest updates:
    ```git pull ```
